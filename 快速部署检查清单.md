## 🚀 Railway 部署步骤（5分钟）

### 第一步：访问 Railway
👉 打开浏览器：https://railway.app

### 第二步：登录
1. 点击右上角 **"Login"**
2. 选择 **"Login with GitHub"**
3. 授权访问

### 第三步：创建项目
1. 点击 **"New Project"**
2. 选择 **"Deploy from GitHub repo"**
3. 找到并点击 **"bbk game"** 仓库
4. 点击 **"Deploy Now"**

### 第四步：等待部署
⏱️ 等待 1-3 分钟，看到 ✅ **"Success"**

### 第五步：生成域名
1. 点击项目卡片
2. 进入 **"Settings"**
3. 找到 **"Networking"** 或 **"Domains"**
4. 点击 **"Generate Domain"**
5. 复制域名（格式：`https://xxxx.up.railway.app`）

### 第六步：配置持久化存储（重要！）
1. 在项目页面，点击 **"Settings"**
2. 找到 **"Volumes"**
3. 点击 **"New Volume"**
4. 挂载路径：`/app`
5. 点击 **"Add"**

### 第七步：测试访问
📱 在浏览器打开你的域名，测试功能！

## ❓ 遇到问题？

### 问题1：找不到仓库
**解决**：在 Railway 点击 "Configure GitHub App"，授权仓库访问权限

### 问题2：部署失败
**解决**：查看 Deployments → Logs，检查错误信息

### 问题3：数据丢失
**解决**：确保配置了 Volume（持久化存储）

### 问题4：域名访问慢
**解决**：Railway 服务器在国外，国内访问可能较慢（可考虑使用国内平台）
