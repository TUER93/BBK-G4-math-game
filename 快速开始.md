## 步骤3: 导入初始数据

### 方法A: 使用管理后台 (推荐)

1. 打开浏览器访问: `http://localhost:3000/admin.html`

2. 点击顶部"📤 批量导入"标签

3. **导入学生:**
   - 打开项目文件夹中的 `示例学生名单.txt`
   - 复制所有内容
   - 粘贴到"批量导入学生"的文本框
   - 点击"导入学生"
   - 看到提示"成功导入 40 个学生!"

4. **导入题目:**
   - 打开项目文件夹中的 `示例题库-大量题目.txt`
   - 复制所有内容
   - 粘贴到"批量导入题目"的文本框
   - 点击"导入题目"
   - 看到提示"成功导入 100 道题目!"

### 方法B: 手动替换文件

直接用示例文件替换原文件即可:
- 用 `示例学生名单.txt` 的内容更新 `students.json` (需调整格式为JSON)
- 用 `示例题库-大量题目.txt` 的内容导入题库

## 常用管理操作

### 添加新学生
1. 访问 `http://localhost:3000/admin.html`
2. 点击"👥 学生管理"
3. 点击"➕ 添加学生"
4. 填写信息并保存

### 添加新题目
1. 访问管理后台
2. 点击"📝 题目管理"
3. 点击"➕ 添加题目"
4. 填写题目、答案、解析、难度

### 查看游戏数据
1. 访问管理后台
2. 点击"📊 用户数据"
3. 查看所有学生的等级、元素等数据

1. 访问管理后台
2. 点击顶部"📦 备份数据"

## 常见问题

### Q: npm install 失败?
A: 使用国内镜像:
```bash
npm config set registry https://registry.npmmirror.com
npm install
```

### Q: 端口已被占用?
A: 修改 `server.js` 中的 `PORT` 变量:
```javascript
const PORT = process.env.PORT || 3001;  // 改为3001或其他端口
```

### Q: 如何添加更多题目?
A: 
1. 推荐使用管理后台批量导入
2. 或编辑 `questions.json` 文件
3. 详见"管理后台使用指南.md"

### Q: 学生数据能修改吗?
A: 可以! 在管理后台中随时增删改

### Q: 如何重置游戏数据?
A: 删除 `data.json` 文件,重启服务器即可
