## 📊 后台功能概览

### 1️⃣ 学生管理（👥）
**功能：**
- ✅ 查看所有学生信息
- ✅ 添加/编辑/删除学生
- ✅ 搜索学生（按姓名或班级）
- ✅ 查看统计数据：
  - 总学生数
  - 班级数量

**操作：**
- 点击 `➕ 添加学生` 添加新学生
- 点击 `✏️` 编辑学生信息
- 点击 `🗑️` 删除学生

---

### 2️⃣ 题目管理（📝）
**功能：**
- ✅ 查看所有题目
- ✅ 添加/编辑/删除题目
- ✅ 按难度筛选（简单/中等/困难）
- ✅ 搜索题目内容
- ✅ 查看统计数据：
  - 题目总数
  - 简单题数量
  - 中等题数量
  - 困难题数量

**题目难度说明：**
- **简单**：正常答题模式
- **中等**：正常答题模式
- **困难**：挑战模式专用

**操作：**
- 点击 `➕ 添加题目` 添加新题目
- 选择难度筛选题目
- 点击 `✏️` 编辑题目
- 点击 `🗑️` 删除题目

---

### 3️⃣ 用户数据（📊）
**功能：**
- ✅ 查看所有玩家游戏数据
- ✅ 查看详细信息：
  - 班级、姓名
  - 当前等级
  - 元素背包（火、水、风、岩、草、雷、冰）
  - 答题统计（总题数、正确率）
  - 已答题目列表

**数据说明：**
- **等级**：玩家当前等级

- **已答题**：已完成的题目 ID 列表

**操作：**
- 点击 `🗑️` 删除用户数据（谨慎操作！）
- 使用搜索框查找特定玩家

---

### 4️⃣ 批量导入（📤）
**功能：**
- ✅ 批量导入学生信息
- ✅ 批量导入题目

#### 导入学生格式（JSON）：
```json
{
  "students": [
    {
      "className": "四(1)班",
      "name": "张三",
      "account": "001"
    },
    {
      "className": "四(1)班",
      "name": "李四",
      "account": "002"
    }
  ]
}
```

#### 导入题目格式（JSON）：
```json
{
  "questions": [
    {
      "question": "计算：25 + 37 = ?",
      "answer": "62",
      "explanation": "25 + 37 = 62",
      "difficulty": "简单"
    },
    {
      "question": "计算：123 × 45 = ?",
      "answer": "5535",
      "explanation": "123 × 45 = 5535",
      "difficulty": "困难"
    }
  ]
}
```

**操作步骤：**
1. 选择"学生导入"或"题目导入"
2. 在文本框中粘贴 JSON 数据
3. 点击"开始导入"
4. 等待导入完成
