============================================
  腾讯云 CloudBase 部署 - 快速开始
============================================

📋 第一步：注册腾讯云（5分钟）
--------------------------------------------
1. 访问 https://cloud.tencent.com
2. 点击"免费注册" → 微信扫码登录（最快）
3. 完成实名认证（必须）
   - 进入控制台会提示认证
   - 选择"个人认证"
   - 填写身份信息，提交

💡 学生可进行学生认证获得更多优惠：
   https://cloud.tencent.com/act/campus


📋 第二步：开通云开发（3分钟）
--------------------------------------------
1. 访问 https://console.cloud.tencent.com/tcb
2. 点击"同意协议并开通"
3. 点击"新建环境"
   - 环境名称：bbk-game
   - 计费方式：按量付费（有免费额度）
   - 地域：上海
4. 点击"立即开通"
5. 等待1-2分钟初始化完成


📋 第三步：开通云托管（2分钟）
--------------------------------------------
1. 在左侧菜单找到"云托管"
2. 点击"开通"
3. 选择"按量付费"
4. 点击"确认开通"
5. 点击"新建服务"
   - 服务名称：bbk-game
   - 点击"确认"


📋 第四步：打包项目（5分钟）
--------------------------------------------
1. 在项目文件夹中，选中以下文件：
   ✅ server.js
   ✅ app.js
   ✅ audio-manager.js
   ✅ index.html
   ✅ styles.css
   ✅ admin.html
   ✅ admin.js
   ✅ admin-style.css
   ✅ package.json
   ✅ package-lock.json
   ✅ questions.json
   ✅ students.json
   ✅ data.json
   ✅ Dockerfile
   ✅ .dockerignore
   ✅ cloudbaserc.json
   ✅ audio 文件夹（所有音频）
   ✅ 所有 .md 和 .txt 文档

2. ❌ 不要选中 node_modules 文件夹！

3. 右键 → 发送到 → 压缩(zipped)文件夹

4. 命名为：bbk-game.zip


📋 第五步：上传部署（5分钟）
--------------------------------------------
1. 在云托管服务页面，点击"新建版本"
2. 选择"本地代码"
3. 上传刚才的 bbk-game.zip 文件
4. 配置参数：
   - 监听端口：80
   - 流水线：Node.js 16
   - 副本个数：1
   - CPU：0.25核
   - 内存：0.5G
5. 点击"开始部署"
6. 等待3-5分钟部署完成


📋 第六步：配置持久化存储（3分钟）
--------------------------------------------
1. 部署完成后，点击"存储"标签
2. 点击"绑定文件系统"
3. 如果没有文件系统，点击"新建文件系统"：
   - 名称：bbk-data
   - 可用区：选择与服务相同
   - 存储类型：通用标准型
   - 容量：10GB
4. 挂载配置：
   - 挂载点路径：/app/data
   - 勾选"自动创建目录"
5. 点击"确认"
6. 等待挂载完成


📋 第七步：获取访问地址（1分钟）
--------------------------------------------
1. 点击"访问配置"标签
2. 复制默认域名（格式如）：
   https://bbk-game-xxxxx.sh.run.tcloudbase.com
3. 在浏览器中访问测试
4. 应该能看到登录页面


✅ 部署完成检查：
--------------------------------------------
□ 能访问首页
□ 能正常登录
□ 能加载题目
□ 能提交答案
□ 音效正常播放
□ 管理后台可访问（/admin.html）
□ 数据能保存（刷新后不丢失）

🆘 遇到问题？
--------------------------------------------
1. 查看部署日志找错误信息
2. 确认监听端口设置为 80
3. 确认没有上传 node_modules
4. 查看详细教程：腾讯云CloudBase部署教程.md
